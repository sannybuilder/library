This command sets get out of jail free for the player. When enabled, if the player is busted, you will not lose any cash after respawning but weapons will still be removed. The get out of jail free works once only, so to allow this effect again you have to enable it through this command again. The value set with this command is saved in block 16 of the save file in GTA III. Although the command does not exist in Vice City and San Andreas, the behavior still exists and can be enabled through other means.
