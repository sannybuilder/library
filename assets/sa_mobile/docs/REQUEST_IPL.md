This command can make a group of objects defined in a binary IPL file to appear in the game. To make it disappear, use REMOVE_IPL. The following is a list of object groups used in the original script.

| Name       | Description                                                |
| ---------- | ---------------------------------------------------------- |
| BARRIERS1  | Los Santos bridge barriers                                 |
| BARRIERS2  | San Fierro bridge barriers                                 |
| CRACK      | Smoke's Crack Palace entrance                              |
| CARTER     | Objects inside Smoke's Crack Factory before being blown up |
| TRUTHSFARM | Truth's crops                                              |
