This command sets the maximum number of vehicles for a save garage to open. Without this command, the default number is 4, since all save garages can only save a maximum of four vehicles. Setting the value below 4 will prevent the garage door from opening for additional vehicles and will display a help message with GXT key `GA_21` when attempting, but the garage door can be forced open and can still save up to four vehicles without any problem. Any values above the range will behave like the default value of 4.
