This command switches the lift camera, which displays a white interference overlay on the screen. The command is used in "G-spotlight" for the elevator camera. Similar in effect and style to SWITCH_SECURITY_CAMERA, the interference will overlay almost everything on the screen but notable exceptions include the clock and some texts like messages and big messages. The scan line will display a darker shade bar that scrolls down the screen and loops back after 62 frames. In Liberty City Stories, the interference command is unused but still works, but there is much less interference and there is no dark bar. The lift camera is switched off when a mission cleanup is processed.
