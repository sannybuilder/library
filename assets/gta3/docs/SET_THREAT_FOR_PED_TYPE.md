This command sets the ped type's hostility to other ped types and will attack the threat ped type on sight. The threat ped type cannot overwrite hardcoded threats but it can overwrite most threats set in the ped.dat file. The threat ped type value is represented as a flag field. The effect of this command can be reset using CLEAR_THREAT_FOR_PED_TYPE. Peds that are already spawned cannot have their hostility changed actively using this command; this command only works on newly spawned peds.
