This command sets the intensity of the riot noise, which is played by the `Law4Riot.mp3` audio file. The riot noise can only be heard at the hardcoded location centered at `x=300.7, y=-322.0, z=12.0` within a radius of `65.0` units. The noise cannot be heard while in a vehicle or in an interior except interior `13`. The intensity is set to `0` when a mission cleanup is processed. The original script called this command during "Riot".
