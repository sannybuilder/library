This command attaches the object to the script path. The object is relocated to the current position of the path by the center of its bounding box so it does not matter how or where the object was created beforehand. The object's position and heading cannot be changed after being attached. Once the first object is attached to the path, the path will immediately start with a speed of 1.0 though that can be changed through SET_OBJECT_PATH_SPEED. This command can only be used a maximum of six times per script path; any more the object will not attach and may cause game instability.
