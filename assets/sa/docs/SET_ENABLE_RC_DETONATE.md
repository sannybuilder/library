This command sets manual detonation of RC vehicles. RC vehicles are created using GIVE_REMOTE_CONTROLLED_CAR_TO_PLAYER or GIVE_REMOTE_CONTROLLED_MODEL_TO_PLAYER, and both commands allow the player to manually detonate the vehicles by default, which can be done by pressing button 17 (attack) . When a mission cleanup is processed, the resulting behavior is equivalent to setting the value of the command to `1`, the default behavior.
